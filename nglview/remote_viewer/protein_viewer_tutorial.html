Aim: Lightweight protein viewer for remote cluster¶

Need:

  • Jupyter notebook
    conda install jupyter notebook

  • pytraj (AmberTools): Used for loading pdb files or trajectories from disk

  • nglview

    pip install git+https://github.com/arose/nglview
  • a modern web browser (Chrome, Firefox ...)

In remote cluster¶

(remote cluster is where you store your data)

  • cd /to/your/working/folder
  • jupyter notebook --no-browser --port=8000

Note: Use can specify any available port number

You will see something similiar to below

$ jupyter notebook --no-browser --port=8000

In your local machine (Windows, Mac laptop, ...)¶

  • Open your terminal and type:

    ssh -N -f -L localhost:8000:localhost:8000 your_account@your_group.rutgers.edu # hit Enter

    Note: replace "your_account@your_group.rutgers.edu" to your real login account

  • Now open web browswer, type "localhost:8000"
